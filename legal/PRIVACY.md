# Privacy Statement

*Last updated: February 2026*

---

## Summary

- Images are processed transiently and **not stored** by default
- We log minimal metadata for service operation
- We do **not** share data with third parties
- We do **not** use your images for training

---

## Image Processing

### Transient Processing

When you upload an image:
1. The image is received by our server
2. Analysis is performed in memory
3. Results are returned to you
4. The image is **deleted immediately** after processing

We do **not** retain copies of uploaded images.

### No Training Use

Your uploaded images are **never** used to:
- Train or improve our models
- Build datasets
- Conduct research
- Any purpose beyond providing you with analysis results

---

## Data We Collect

### Request Metadata

For service operation and security, we log:

| Data | Purpose | Retention |
|------|---------|-----------|
| Request timestamp | Service monitoring | 30 days |
| Request ID | Error tracking | 30 days |
| File size | Rate limiting | 30 days |
| Response code | Service health | 30 days |
| Processing time | Performance monitoring | 30 days |

### What We Do NOT Collect

We do **not** collect or log:
- Image content or pixels
- Image filenames
- User identity information
- IP addresses (anonymized if collected)
- Location data
- Device identifiers

## Third-Party Sharing

We do **not** share any data with third parties:
- No advertising partners
- No analytics services (beyond basic server logs)
- No data brokers
- No social media platforms

---

## Data Security

- All connections use HTTPS encryption
- Image data is processed in isolated memory
- No persistent storage of image content
- Server logs are access-controlled
